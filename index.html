📖 Introduction

In this task you'll demonstrate your Level 2 Roblox knowledge and your ability to guide students as they work on more advanced projects. Download the project file below, open it in Roblox Studio, and complete all the assignments.

Assignment 1 — Setup & Building
1
Download the project file and open it in Roblox Studio.
📦 Download Project File →
2
Using a ClickDetector, create a working shop button that gives the player a Tool from ServerStorage when clicked. The tool must appear in the player's Backpack.
3
Create an NPC that follows a series of at least 3 waypoints in a loop. The NPC must move continuously and reset to the first waypoint after reaching the last.
4
Add a GUI countdown timer (60 seconds) that is visible on-screen from the start of the game. When it reaches zero, display a "Time's Up!" message in red.
Assignment 2 — Scripting
1
The project contains scripting errors. Identify and fix each one:
1.1
When a player collects a coin, a pickup sound does not play. Locate the bug and fix it.
1.2
When a player collects their 10th coin, a special milestone sound should play instead of the regular one — but it doesn't. Fix it.
2
Implement DataStore saving: the player's coin count must persist between sessions. When they rejoin, their coins should load back to where they left off.
